ทบทวนความรู้เดิม

ก่อนเข้าสู่เรื่องใหม่ ลองทายผลลัพธ์ของโค้ดที่เคยเรียนผ่านมาแล้วกันก่อน

1) โค้ดนี้ใช้โครงสร้างควบคุมแบบใด?

recap1.py
1for i in range(5):
2    print(i)
A คำสั่งเลือกทำ (if-else)
B คำสั่งวนซ้ำแบบกำหนดจำนวนรอบ (for)
C คำสั่งวนซ้ำแบบมีเงื่อนไข (while)
D คำสั่งรับค่าจากผู้ใช้

2) จากโค้ดต่อไปนี้ เงื่อนไข score >= 50 จะทำงานอย่างไร?

recap2.py
1score = 65
2if score >= 50:
3    print("ผ่าน")
4else:
5    print("ไม่ผ่าน")
A พิมพ์ "ไม่ผ่าน" เพราะ 65 น้อยกว่า 50
B พิมพ์ "ผ่าน" เพราะ 65 มากกว่าหรือเท่ากับ 50
C เกิดข้อผิดพลาด เพราะเขียนเงื่อนไขผิด
D พิมพ์ทั้งสองบรรทัด

ขั้นที่ 2 · สำรวจและค้นหา

ทำความรู้จัก while loop

while คือคำสั่งที่ทำงานซ้ำไปเรื่อย ๆ ตราบใดที่เงื่อนไขยังเป็นจริง ต่างจาก for ที่กำหนดจำนวนรอบไว้ล่วงหน้า

โครงสร้างคำสั่ง

structure.py
1count = 1          # ① ตั้งค่าเริ่มต้น
2while count <= 5: # ② เงื่อนไข — เป็นจริงจึงวนต่อ
3    print(count) # ③ คำสั่งที่ทำซ้ำ
4    count += 1     # ④ เปลี่ยนค่าตัวแปร (ขาดไม่ได้!)

ทดลองไล่ค่าไปทีละรอบ (Trace Table)

กดปุ่มเพื่อดูว่าตัวแปร count เปลี่ยนแปลงอย่างไรในแต่ละรอบ

สาระการเรียนรู้เสริม

คลังจับผิดสุดฮิต

ก่อนไปแข่งขันในเกม Bug Hunter มาทำความรู้จักข้อผิดพลาดที่นักเขียนโปรแกรมมือใหม่พบบ่อยที่สุดกันก่อน

อาการสาเหตุที่พบบ่อยวิธีแก้
Infinite Loop โปรแกรมค้าง ไม่หยุดวนซ้ำลืมเปลี่ยนค่าตัวแปรที่ใช้ในเงื่อนไขตรวจสอบทุกครั้งว่ามีคำสั่งเพิ่ม/ลดค่าตัวแปรอยู่ใน loop
SyntaxError โปรแกรมรันไม่ได้ลืมใส่เครื่องหมาย : ท้ายเงื่อนไขตรวจการใช้ : หลัง while เสมอ
IndentationError / ทำงานผิดจังหวะคำสั่งภายใน loop เยื้องไม่ตรงกันจัดย่อหน้า (indent) ให้สม่ำเสมอ 4 ช่องเสมอ
วนซ้ำเกิน/ขาด ผลลัพธ์ไม่ตรงตามต้องการใช้ != แล้วค่าตัวแปรกระโดดข้ามค่าที่ต้องการพอดีใช้ < หรือ <= แทน != เมื่อไม่แน่ใจว่าค่าจะตรงพอดี

โจทย์ที่ 1 — นับถอยหลัง

โจทย์ที่ 2 — ผลรวมเลขคู่

เขียนโปรแกรมหาผลรวมของเลขคู่ตั้งแต่ 2 ถึง 20 โดยใช้ while

โจทย์ที่ 3 — ทายตัวเลข

กำหนดตัวแปรลับ secret = 7 ให้เขียนลูป while ที่ถามผู้ใช้ซ้ำไปเรื่อย ๆ จนกว่าจะทายถูก

💡 คำใบ้: อย่าลืมตรวจสอบว่าทุกลูปของท่านมีทาง "จบ" เสมอ ไม่เช่นนั้นโปรแกรมจะกลายเป็น Infinite Loop เหมือนในหัวข้อที่แล้ว!
ทำเสร็จแล้ว? ให้ตัวแทนกลุ่มเตรียมนำเสนอโค้ดหน้าชั้นเรียน แล้วไปลุยภารกิจล่าบั๊กกันเลย!
